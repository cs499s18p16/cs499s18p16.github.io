Product Requirements


Features (in priority order)

  • Database
    • Use MySQL database, which is an open source search-query language database management system, to store the login information of each employee on the Scrum team, as well as each week's productivity statistics
    • Specifically, be able to store a team member's individual capacity in points, the point values for sprint stories, and from there be able to calculate total capacity for each sprint (essential). If time allows, also be able to store and manipulate data necessary to calculate sprint velocity (stretch goal).
  • Web Application
    • The site will be hosted via Amazon Web Services.
    • This site will house all the databases and will be accessible by all qualified employees using a secure login unique to each of them
    • GUI for Windows and Mac OS
    • HTTPS will be used for secure login
  • User Interface
    • The web application and database backend UI requirements are not yet set in stone. Details to come.
